What is pbckcode ?

This is a CKEditor plugin that provides an IDE-like features and creates the good syntax to load your syntax highlighter. The IDE-like features are possible owing to the ACE Editor plugin.

Why pbckcode ?

This plugin comes from my personnal need. I am becoming a blogger and I want to write code articles and tutorials. However I did not found any decent plugins to add code blocks into posts so I decided to create mine.

Whose for?

Writer, bloggers, coders, developers, everbody !
